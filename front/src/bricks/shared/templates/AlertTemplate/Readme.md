### Import

```js static
import { AlertTemplate } from 'pretto-ui'
```

### Usage

```html
<AlertTemplate />
```

### Examples

#### Basic:

```jsx
<AlertTemplate
  title="Alert Template"
  description="Lorem ipsum dolor sit amet, consectetur adipisicing elit. Perferendis, ab."
  alert="Lorem ipsum dolor sit amet, consectetur adipisicing elit. Necessitatibus ipsa repellendus illo!"
>
  <Card variant="white" zDepth={2}>
    <Content>
      <p>
        Lorem ipsum dolor sit amet, consectetur adipisicing elit. Iste nemo quidem ducimus modi obcaecati, doloremque
        eligendi ut excepturi distinctio beatae impedit aliquid nulla non totam quaerat minima? Facilis, perferendis,
        laudantium!
      </p>
    </Content>
  </Card>
</AlertTemplate>
```
