### Import

```js static
import { Sequence } from 'pretto-ui'
```

### Usage

```html
<Sequence>
  <header>I am an alert</header>
  <p>With some text in it</p>
</Sequence>
```

### Examples

#### Basic:

```jsx
<Sequence>
  <p>I am an Sequence child.</p>
  <p>me too.</p>
  <p>me too!</p>
  <p>I am the fastest child.</p>
</Sequence>
```
